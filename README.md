# GCN-CNN
TCGA cases used in this study are listed in file *.event_name.txt
You can download gene expression data from  https://www.cancer.gov/tcga
For each case in this study, index and clinic information are as shown in *.index.txt, in which first column presents a case is kept or filtered, second column presents a case's label, third column presents a case's clinic time point.
X_gc_noweight.txt is each gene's weight tabel generated by GCN.


To repeat experiment, first download TPM data from TCGA database, then run data_pre.py to generate data, then CNN.py to check model's classification performance.
